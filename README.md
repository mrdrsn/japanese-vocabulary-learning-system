# Интеллектуальная система изучения японской лексики

## Описание проекта

Проект посвящен проектированию интеллектуальной системы для контекстно-ориентированного изучения японской лексики.

Система ориентирована на туристов, которые не владеют японским языком, но хотят быстро освоить лексику и типовые фразы для распространенных коммуникативных ситуаций: кафе, магазин, регистрация в отеле, транспорт и другие сценарии.

## Цель проекта

Разработать проект интеллектуальной системы, которая позволяет изучать японскую лексику не изолированно, а в контексте конкретных коммуникативных сценариев.

## Выполненные этапы проекта

- анализ предметной области;
- формализация функциональных требований;
- описание пользовательских сценариев;
- проектирование архитектуры системы;
- проектирование реляционной базы данных;
- проектирование графовой базы знаний;
- моделирование экранных форм;
- описание принципа получения лексики по выбранному сценарию.

## Основные функции системы

- отображение перечня коммуникативных сценариев;
- выбор пользователем сценария;
- подбор лексико-фразовых единиц по выбранному сценарию;
- отображение ромадзи и перевода;
- открытие карточки лексической единицы;
- воспроизведение аудиоматериала;
- запуск карточного режима изучения;
- формирование тренировочных заданий на восприятие на слух.

## Архитектура системы

Архитектура системы включает:

- пользовательский интерфейс;
- backend-логику приложения;
- реляционную базу данных для хранения пользовательских и служебных данных;
- графовую базу знаний для хранения сценариев, ролей, шаблонов фраз и лексических единиц.

Подробнее: [docs/architecture.md](docs/architecture.md)

## База данных и база знаний

В проекте используются два типа хранилищ:

- реляционная база данных – для хранения пользователей, прогресса, результатов тренировок;
- графовая база знаний – для описания связей между коммуникативными сценариями, шагами сценария, ролями, структурными шаблонами и лексическими единицами.
