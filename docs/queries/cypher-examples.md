# Примеры Cypher-запросов

В данном разделе приведены примеры Cypher-запросов для работы с графовой базой знаний, используемой в системе.

База знаний содержит структурные шаблоны, слоты, лексические единицы и коммуникативные намерения, что позволяет формировать фразы в зависимости от контекста использования.

---

## 1. Формирование предложений на основе шаблонов

Запрос возвращает таблицу с:

- шаблоном фразы;
- лексической единицей;
- предложением на ромадзи;
- переводом предложения.

```cypher
MATCH (t:StructuralTemplate)-[:HAS_SLOT]->(s:Slot)-[:CAN_BE_FILLED_BY]->(l:LexicalUnit)

RETURN
t.pattern AS template,
l.romaji AS word,
replace(t.pattern, "[X]", l.romaji) AS sentence_romaji,
replace(t.translation, "[X]", l.translation) AS sentence_translation
ORDER BY template, word
```
Назначение

Используется для генерации вариантов фраз на основе структурных шаблонов и доступной лексики.

## 2. Получение предложений с коммуникативным намерением

Запрос возвращает:

- сформированное предложение (ромадзи);
- перевод предложения;
- коммуникативное намерение.

```cypher
MATCH (t:StructuralTemplate)-[:HAS_SLOT]->(s:Slot)
MATCH (s)-[:CAN_BE_FILLED_BY]->(l:LexicalUnit)
MATCH (t)-[:EXPRESSES]->(ci:CommunicativeIntent)

RETURN
replace(t.pattern, "[X]", l.romaji) AS sentence_romaji,
replace(t.translation, "[X]", l.translation) AS sentence_translation,
ci.name AS communicative_intent
ORDER BY sentence_romaji
```
Назначение

Позволяет связать языковую конструкцию с её коммуникативной функцией (например: просьба, вопрос, уточнение).

## 3. Визуализация графовой структуры

Запрос возвращает полный граф базы знаний:

```cypher
MATCH (n)-[r]->(m)
RETURN n, r, m;
```
Назначение

Используется для визуального анализа структуры базы знаний:

- просмотр всех узлов и связей;
- проверка корректности модели;
- отладка связей между сущностями. 
## 4. Удаление узла вместе со связями

Запрос удаляет узел вместе со всеми его связями:

```cypher
MATCH (n:Slots)
DETACH DELETE n;
```
Назначение

Используется для:

- чистки данных;
- удаления ошибочно созданных узлов;
- обновления структуры графа.
